# Doqlo Demo Assets

Official demo assets for Doqlo product walkthroughs and PDF workflow examples.

This repository contains sample files used in Doqlo tutorials, product demos, and public walkthrough videos.

## What's included

This repository may include:

- sample CSV files
- `.doqlo` project files
- preview output examples
- batch export examples
- notes for specific walkthroughs

## Current demos

### Bulk Fill — Fill a fillable PDF form from CSV

Assets for the Doqlo Bulk Fill walkthrough that shows how to:

- open a fillable PDF form
- map form fields to CSV columns
- save and reopen a `.doqlo` project
- export a preview
- export a batch ZIP of filled PDFs

Files in this demo folder include:

- `demo.csv`
- `demo.doqlo`
- `preview-example.pdf`
- `batch-export-example.zip`

## Official source documents

When a demo uses a public form from an external source, the official source link is referenced in the related notes or video description.
