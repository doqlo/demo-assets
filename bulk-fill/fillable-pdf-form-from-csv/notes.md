# Fill a fillable PDF form from CSV

This folder contains the demo assets used in the Doqlo Bulk Fill walkthrough for filling a fillable PDF form from CSV.

## Files

- `i-9-demo.csv` — sample CSV data used in the walkthrough
- `i-9-demo.doqlo` — saved Doqlo project with field-to-column mappings
- `preview-example.pdf` — example preview export
- `batch-export-example.zip` — example batch export output

## Source form

Official USCIS Form I-9 source:
https://www.uscis.gov/sites/default/files/document/forms/i-9.pdf

## Notes

All data in this demo is sample data for product demonstration only.
